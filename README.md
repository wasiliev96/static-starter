# Static Starter
New static site template

---
## Includes
### CSS
Featues:
- autoprefixer
- postcss-normalize - canculate needed resets by browserslist config
- sass
- browser-sync
- font-magician(optional) - connect fonts like a pro :)
- critical - calculate critical(first-screen) code and put it inline

### HTML
Features:
- Pug

### Images
Features:
> Creates separate `.webp` files in dist
- gulp-imagemin
- imagemin-jpeg-recompress
- imagemin-pngquant
- gulp-webp

#### Sprites
Features:
- gulp-svgstore
- gulp-imagemin(svgo)

### Javasript
Features:
- Babel
- Webpack

---
---
## Install
`npm i` or `yarn i`

## Dev mode
`yarn dev` or `npm run dev`

## Build 
`yarn build` or `npm run build`
